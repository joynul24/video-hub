Video Hub is a dynamic and responsive video platform where users can explore videos by category, search by title, sort by views, and view detailed information in a modal.

## ‚ú® Features

- **Responsive Navbar**
  - Logo on the left
  - Search box in the center
  - Sort button on the right
  - Bottom border for clean UI

- **Dynamic Category Section**
  - Categories loaded from API
  - Active button state for better UX
  - Clicking a category loads videos for that category

- **Video Cards**
  - Videos fetched from API
  - Verified badge shown if author is verified
  - Views and posted time displayed
  - Details button opens a modal with full video info

- **Fallback UI**
  - Displays a friendly error icon and message if no videos are available in a category

- **Search Functionality**
  - Search videos by title
  - Deactivates category filter when searching

- **Sort by Views**
  - Sorts videos in descending order based on views
